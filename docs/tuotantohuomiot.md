# Tuotantohuomiot

Alla ovat ne asiat, jotka maarittelyssa eivat olleet viela aivan lukittuja, mutta jotka kannattaa paattaa ennen varsinaista kayttoonottoa.

## Tarkeimmat puuttuvat paatokset

- Kayttooikeudet: kayttavatko kaikki samaa tunnusta vai kirjautuuko jokainen tyontekija omilla tunnuksillaan.
- Tulostus: tulostetaanko yksittainen tilaus, paivakohtainen lista vai tuotekohtainen raportti suoraan tiskitulostimelle.
- Puhelinnumeron muoto: sallitaanko vain suomalaiset numerot vai myos kansainvaliset.
- Muokkausloki: tarvitaanko tieto kuka muutti tilausta ja milloin.
- Arkistointi: kuinka pitkaksi aikaa vanhat sesonkitilaukset pidetaan aktiivisessa nakymassa.
- Peruutukset: tarvitaanko erillinen peruttu-tila vai riittaako poistaminen.
- Noutoikkuna: riittaako pelkka noutopaiva vai tarvitaanko myos kellonaika.
- Offline-kaytto: halutaanko sovelluksen toimivan hetken myos ilman verkkoyhteytta.

## Suositellut seuraavat lisat

- Supabase Auth henkilokunnalle.
- Audit log -taulu tilausten muutoksille.
- Oma tulostusnakyma paivan tilauksille.
- Sesonkikohtaiset tuotelistat, jotta joulun ja juhannuksen valikoimat voi hallita erikseen.
- CSV- tai PDF-vienti raportteihin.
